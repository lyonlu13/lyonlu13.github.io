  1. Give n, a positive integer, determine if n is the sum of divisors, that is, if n is the sum of all t such that 1 ≤ t < n and t divides n [Ch1.3]
    //if true return 1, otherwise return 0
int determine(int n){
	int sum=0;
	for(int i=1;i<n;i++)
		if(n%i==0) sum+=i;
	return sum==n?1:0;
}
